# Dark Chess

A modern chess web application built with JavaScript and Rsbuild.

## Overview

This project is currently in early development, creating a chess platform with modern web technologies.

## Tech Stack

- **JavaScript** - Core programming language
- **Rsbuild** - Build tool and dev server
- **Rspack** - Fast bundler
- **Biome** - Linting and formatting

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

```bash
npm install
```

### Development

```bash
# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Code Quality

```bash
# Lint code
npm run lint

# Format code
npm run format
```
